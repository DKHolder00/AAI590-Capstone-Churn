# AAI590 Capstone Project: Customer Churn Prediction

Author: D. Keith Holder

## Overview
This project develops a machine learning pipeline to predict customer churn using multiple modeling approaches, including:

- Logistic Regression
- Decision Tree
- Ensemble Models (Random Forest, AdaBoost, Gradient Boosting)
- Multi-Layer Perceptron (MLP)
- Stacked Ensemble Model

## Notebook
The main notebook contains:
- Data preprocessing
- Feature engineering
- Model development
- Evaluation and comparison
